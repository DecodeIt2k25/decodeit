Day 1 – Introduction to HTML

✅ HTML (HyperText Markup Language)

HTML is not a programming language; it is a markup language.

Markup refers to the tags used inside a document to define the structure.

HTML defines only the structure of a webpage. It does not control style, design, or functions.

Each HTML element tells the browser how to display different parts of the page — such as headings, images, links, and paragraphs.

HTML has a boilerplate — a basic structure used in every webpage.

✅ Basic Structure of an HTML Document


<!DOCTYPE html>
<html>
<head>
  <title>Enter Title</title>
</head>
<body>
  <p>Your paragraph goes here</p>
</body>
</html>

✅ Difference Between Webpage and Web Browser

Webpage:

  • It is a file saved with a .html extension.
  • It contains elements like text, images, and links.
  • Example: The HTML structure shown above.

Web Browser:

  • Software used to open and view a webpage.
  • Examples: Chrome, Microsoft Edge, Brave, Firefox.
  • Without a browser, we cannot run or view a webpage.

✅ Conclusion

From this introduction, you can understand the difference between a webpage and a web browser. You have also learned the basic elements of HTML and how every webpage starts with the same structure.
